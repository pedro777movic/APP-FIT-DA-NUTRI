# **App Name**: Nutri's Recipe Vault

## Core Features:

- Recipe Organization: Organize recipes into categories: 'Café da Manhã,' 'Sobremesas Zero,' 'Bolos,' and 'Airfryer.'
- Local Recipe Storage: Store recipes locally using JSON, IndexedDB, or LocalStorage for offline access.
- Recipe Addition: Add recipes manually via copy and paste directly into the corresponding section.
- Recipe Search: Implement a real-time search functionality to filter recipes by name, operating offline.
- Enhanced Instructions: Employ an AI tool to enhance succinct preparation steps, and provide detailed instructions, adding clarity, without altering ingredient or nutritional information.
- User Authentication: Local email/password-based authentication for each section: Breakfast ('cafe@fit@.com', 'CAFEFIT1'), Desserts ('sobremesas@zero', 'ZEROCAFE2'), Cakes ('bolos@semculpa.com', 'BOLOFIT3'), Airfryer ('Airfryer@proteica', 'PROTEINAFIT4').
- Card-Based Recipe Display: Display recipes as cards with a small illustrative image or icon, placed to the left of the recipe title.

## Style Guidelines:

- Primary color: Dark green (#1F3D2B) to convey health and nature.
- Background color: Very light green (#F0F4EF) for a clean, subtle backdrop.
- Accent color: A desaturated yellow-green (#A8B886) for highlights and interactive elements.
- Headline font: 'Poppins' (sans-serif) for a modern and readable headline.
- Body font: 'PT Sans' (sans-serif) is for readability for longer descriptions, and pairs well with Poppins.
- Simple, intuitive icons for recipe categories and actions.
- Mobile-first design with large, clear buttons, cards, and a fluid vertical scroll. 100% responsive layout. Meaningful spacing, soft shadows, and rounded edges for a premium feel.
- Subtle and brief animations to enhance user interaction without affecting performance.
